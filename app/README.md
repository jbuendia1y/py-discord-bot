# Bot App

## Configura la app

### Crea el archivo .env

```
BOT_TOKEN= XXXX - El token del Bot
PY_ENV= development o production
API_URL=http://localhost
```

### Monta el pipenv

```bash
pipenv shell
```

### Instala las dependencias

```bash
pipenv install
```

### Inica la app

```bash
pipenv run start
```
